# Cloud Cost Optimization Analysis

This project reviews a hypothetical AWS small-scale cloud environment and identifies practical ways to reduce monthly infrastructure costs.

The analysis focuses on:
- EC2 rightsizing
- Compute Savings Plans
- Scheduled shutdown of idle resources
- S3 lifecycle policies
- RDS gp2 to gp3 migration

The report includes estimated savings, implementation considerations, and operational recommendations for improving cloud cost efficiency.
